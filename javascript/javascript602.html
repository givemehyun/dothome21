02. 문자열 메서드 : 문자열 길이 구하기

{

    const currentStr2 = str2.length;

번호 기본 값 메서드 결과값 
{

    const currentStr2 = str2.toLowerCase();

    document.querySelector(".sample03_R2").innerHTML = currentStr2;
}
